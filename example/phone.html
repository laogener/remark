1、box-shadow:h-shadow v-shadow blur spread color position；
h-shadow(offset-x):必需。水平阴影的位置，正值在右，负值在左。
h-shadow(offset-y):必需。垂直阴影的位置，正值在上，负值在下。
blur:可选。阴影模糊半径。模糊数值越大，尖锐度越小，阴影越朦胧和模糊，负值不允许，会被处理成0.
spread：可选。阴影的尺寸。正值会在元素的各个方向按指定的数值延伸阴影。负值会使阴影收缩的比元素本身尺寸还小。默认值0会让阴影伸展的和元素的大小一样。
color：可选。阴影的颜色。
position:可选。outset（默认外部阴影），inset内部阴影。
多重阴影：box-shadow属性能在单个元素上接受多个阴影。每个阴影通过逗号分隔的box-shadow属性列表来加载。
div{
bos-shadow:20px 20px 10px 0px rgba(0,0,0,0.5) inset,
20px 20px 10px 0px rgba(0,0,0,0.5);
}

圆形阴影：box-shadow属性的边界半径是通过该元素的border-radius属性来控制的。

2、user-modify属性，用来控制用户能否对页面文本进行编辑。与标签的contentEditable属性类似。

-webkit-user-modify: read-only | read-write | read-write-plaintext-only

read-only 内容只读。

read-write 内容可读写。

read-write-plaintext-only 内容可读写，但粘贴内容中的富文本格式（如文本的颜色、大小，图片等）会丢失。内容类似于以纯文本显示。

-webkit-tap-highlight-color:苹果手持设备的浏览器属性，意思是：点击它，它就会出现一个半透明的灰色背景

a,button,input,textarea{

-webkit-tap-highlight-color: rgba(0,0,0,0) ;

-webkit-user-modify:read-write-plaintext-only;

}

a,img,button,input,textarea,span,ul,li,div,p{-webkit-tap-highlight-color:rgba(255,255,255,0);}

input[type=button],textarea{-webkit-appearance:none;}

3、在谷歌浏览器会默认记住账号，而记住账号之后其input的背景会变成黄色，解决的办法如下：

方法一：直接用css的内阴影来覆盖黄色，代码如下：

input:-webkit-autofill {

-webkit-box-shadow: 0 0 0px 1000px white inset;

}
